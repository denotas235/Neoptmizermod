# 🚀 Nexus Optimizer Mod
**Otimizações avançadas para Minecraft Java Edition, compatível com Sodium e focado em GPUs móveis (TDBR).**

---

## 📌 **Recursos**
✅ **Compatível com Sodium** – Não conflita com as otimizações do Sodium.
✅ **Occlusion Culling** – Reduz *overdraw* em chunks ocultos.
✅ **Batch Rendering** – Minimiza *draw calls* para melhorar o FPS.
✅ **TDBR-Friendly** – Shaders otimizados para GPUs móveis (Mali Bifrost/Valhall).
✅ **Texturas ASTC** – Suporte a compressão de texturas para dispositivos móveis.

---

## 📥 **Como Instalar**
1. Baixe o `.jar` da seção [Releases](https://github.com/denotas235/Neoptmizermod/releases).
2. Coloque o arquivo na pasta `mods/` do seu Minecraft.
3. **Requerimentos**:
   - Minecraft **1.21.1**
   - [Fabric Loader](https://fabricmc.net/)
   - [Fabric API](https://www.curseforge.com/minecraft/mc-mods/fabric-api)
   - **Recomendado**: [Sodium](https://modrinth.com/mod/sodium) (para melhor performance)

---

## ⚙️ **Configurações**
O mod adiciona opções no menu de configurações do Minecraft:
- **Occlusion Culling**: Liga/desliga a detecção de chunks ocultos.
- **Texturas ASTC**: Força compressão de texturas para GPUs móveis.
- **TDBR Shaders**: Ativa shaders otimizados para Tile-Based Deferred Rendering.
- **Tile Size**: Tamanho dos *tiles* (16x16, 32x32, 64x64).
